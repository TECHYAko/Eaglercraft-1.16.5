What's new in u2-beta

  • Major FPS improvements — smoother gameplay, especially on lower-end hardware
  • New WASM-GC engine — smaller download, faster boot times
  • Real multiplayer — connect to real Minecraft servers
  • Shared Worlds — invite friends into your singleplayer world over the internet, no dedicated server needed
  • Custom skins & capes — now show correctly for everyone in Shared Worlds
  • Chat fixes — in-game chat and join/leave messages now work reliably

About u1-beta

  • The original browser release of this port. Kept available for compatibility and testing.
  • Uses the older classic engine build (not WASM-GC) — larger download, slower boot than u2-beta.
